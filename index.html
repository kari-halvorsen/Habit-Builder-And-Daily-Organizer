About this project

A personal productivity dashboard I built to replace a fragmented workflow across Trello, a habit tracker, and Google Calendar. Everything lives in one place, organised around five life goal categories: Health, Financial Freedom, Relationships, Mental Health, and Hobbies.

The live version connects directly to Google Calendar via Google Apps Script, so events load automatically every time the page opens — no syncing needed. This demo uses anonymised placeholder data so you can try everything interactively.

Features: 21-day rolling Kanban board, AI-powered task categorisation (try typing a task below), daily habit tracking with counter boxes, fireworks on completion, today's wins archive, weekly goal progress tracking, life balance breakdown chart, and a move-to-day modal for card management.
